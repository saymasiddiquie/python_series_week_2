What is a Variable?

Think of a variable as a labelled box where you can store data. The label is the variable name, and the contents are the value.

  • Creation: Python creates the variable the moment you assign a value to it.
  • Case Sensitive: age and Age are different variables.
  • Dynamic: You don't need to declare the type. Python figures it out!
main.py
score = 100
player_name = "Rahul"
print(player_name)

Standard Data Types

Python has several built-in types. Here are the main ones you need for Week 2.

int
Whole numbers without decimals.
x = 5
float
Numbers with decimal points.
pi = 3.14
str
Text enclosed in quotes.
name = "GFG"
bool
True or False values.
is_active = True
