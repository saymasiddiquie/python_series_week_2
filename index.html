What is a Variable?

A variable is a named location in computer memory. It's like a box with a label where you can store data to use later.

main.py
score = 100
player_name = "Rahul"
print(player_name)

Rules for Naming Variables

Python is strict about how you name your variables. Follow these rules to avoid errors!

Valid (Allowed)

  • Start Must start with a letter (a-z, A-Z) or an underscore (_).
  • Chars Can contain letters, numbers, and underscores.
  • Case Case Sensitive! age, Age, and AGE are three different variables.

Good Examples:

my_var = 10 _hidden = 5 var2 = "Hello"

Invalid (Not Allowed)

  • Start Cannot start with a number.
  • Space Cannot contain spaces. Use underscores instead.
  • Special No special symbols like @, $, %, -.
  • Reserved Cannot use Python keywords (e.g., class, if, for, import).

Bad Examples:

2myVar = 10 (Starts with number) my var = 5 (Has space) my-var = "No" (Has hyphen)

Standard Data Types

Python automatically detects the type of data you store.

int
Whole numbers without decimals.
x = 5
float
Numbers with decimal points.
pi = 3.14
str
Text enclosed in quotes.
name = "GFG"
bool
True or False values.
is_active = True
