The Basics: If, Elif, Else

Think of your code as a river. Normally, it flows straight down. Conditional statements allow you to divert the flow based on logic.

if

The Guard

Checks a condition. If it's True, the code inside runs. If False, it skips.

elif

The Alternative

Short for "Else If". Only checked if the previous if failed.

else

The Fallback

Catches everything else. No condition needed. Runs if everything above fails.

Important: Indentation

Python uses indentation (spaces) to know what code belongs inside the `if` block. Without it, you get an error!

Mastery Quiz

Prove your knowledge of conditionals. 10 questions to test if you are ready for the next level.

1. Which keyword is used to start a conditional statement?

Correct! `if` is the keyword used to check a condition.

2. What does elif stand for?

Correct! `elif` is Python's shorthand for "else if".

3. Which operator checks if two values are equal?

Correct! `==` compares values. `=` assigns values.

4. What is the result of: True and False?

Correct! For `and`, BOTH sides must be True.

5. What does the else block do?

Correct! `else` is the catch-all fallback.

6. How does Python know which code is inside the if?

Correct! Python relies on correct indentation.

7. What is the result of: not True?

Correct! `not` inverts the boolean value.

8. Can you have an if without an else?

Correct! `else` is optional.

9. What is the result of: True or False?

Correct! For `or`, only ONE side needs to be True.

10. Which is a valid boolean value in Python?

Correct! Python Booleans are capitalized: `True` or `False`.
